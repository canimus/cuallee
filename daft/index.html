    is_complete

    This check is the most popular. It validates the completeness attribute of a data set. It confirms that all fields contain values different of null.

    is_complete 
    import daft
import numpy as np
from cuallee import Check, CheckLevel

df = daft.from_pydict({"id": np.arange(10)})
check = Check(CheckLevel.WARNING, "CompletePredicate")
check.is_complete("id")

# Validate
check.validate(df)

    is_unique

    This check verifies that the number of distinct values for a column is equal to the number of rows of the dataframe.

    is_unique 
    import daft
from cuallee import Check, CheckLevel

df = daft.from_pydict({"id": [10, 20, 30]})
check = Check(CheckLevel.WARNING, "IsUnique")
check.is_unique("id")

# Validate
check.validate(df)
